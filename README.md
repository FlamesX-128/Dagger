# **Ramen's Bot**
## **English:**
It is a bot focused on roles and defense. 

### **Commands:**

    !> = prefix depending on the configuration.
    [] = required argument.
    () = optional argument depending on the configuration.
    <> = optional argument

    !>kick          [user]   (reason)
    !>warn          [user]   (reason)
    !>nuke          [amount]
    !>add-role      [user]   [role]
